# calm-soul

A small project to help users relax and rewind their thoughts by recalling peaceful memories and listening to calming audio.

## Purpose
This repository contains the calm-soul website/app: a place to play relaxing audio, browse nostalgic prompts, and unwind.

## Suggested features
- Simple landing page with calming visuals and audio player
- Collection of short guided prompts and background tracks
- Dark/light theme with smooth transitions
- Local favorites/bookmarks
- Optionally: small backend to track user preferences (optional)

## Getting started (static site)
1. Clone the repo
   ```
   git clone https://github.com/bilalseif70/calm-soul.git
   cd calm-soul
   ```
2. Open `public/index.html` in your browser.

## Getting started (React / Vite)
1. Install Node.js (>=16)
2. Install dependencies:
   ```
   npm install
   ```
3. Start dev server
   ```
   npm run dev
   ```

## Structure
- `public/` - static files (audio, images)
- `src/` - source code
- `docs/` - documentation and design notes
